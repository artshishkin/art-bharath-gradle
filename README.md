####  Section 3: Gradle in action

#####  7. First Gradle Project

-  `gradle init`
    - 1 - basic
    - 1 - Groovy
-  `gradle tasks`    
-  add `firstTask` to build.gradle
-  `gradle tasks --all`
   -  will see `firstTask`
-  Execute task
   -  `gradle firstTask`
   -  **or** short form
   -  `gradle fT`
-  settings.gradle
   -  we can change rootProject.name = 'first-project'
-  run wrapper
   -  `./gradlew fT`
